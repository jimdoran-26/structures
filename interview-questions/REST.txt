-app sends request, server sends response
-servlet sends in XML/JSON
-send a java object in JSON

what is API?

What is REST API?
-sending state of an object
-representational state of an object (as JSON)
-send request as url
-what you are fetching is a resource

How REST works?
-fetching data as it is
-don't care about what is happening on server

    Read   = GET
    Update = PUT
    Delete = DELETE
